# Banker's Algorithm Simulator

A web-based simulator that demonstrates the Banker's Algorithm used in Operating Systems for deadlock avoidance.

## 🚀 Features
- Dynamic input for processes and resources
- Allocation and Maximum matrix input
- Available resources input
- Safe sequence detection
- Step-by-step execution display
- Clean and responsive UI

## 🛠️ Technologies Used
- HTML
- CSS (Tailwind CSS)
- JavaScript

## 📌 How It Works
1. Enter number of processes and resources
2. Fill Allocation, Max, and Available matrices
3. Click "Next"
4. System calculates and displays:
   - Execution steps
   - Safe sequence (if exists)

## 📊 Example Output
Safe Sequence:
P1 → P3 → P4 → P0 → P2

## 🎯 Purpose
This project helps understand how deadlocks can be avoided using the Banker's Algorithm.
